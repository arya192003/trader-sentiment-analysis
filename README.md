# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze how market sentiment (Fear/Greed) impacts trader behavior and performance.

## Methodology
- Data cleaning and preprocessing
- Feature engineering
- Merging datasets
- Behavioral analysis

## Key Insights
- Traders take larger positions during Greed periods
- PnL volatility is higher during Fear periods
- Larger trades carry higher risk

## Strategy Recommendations
- Reduce trade size during Fear periods
- Use trend-following strategies during Greed
- Maintain disciplined position sizing

## Notes
- Synthetic time index was used due to timestamp limitations
